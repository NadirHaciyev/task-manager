# Task Manager

### How to run:

- Download the project

```bash
git clone https://github.com/NadirHaciyev/task-manager.git
cd task-manager
```

- Run client

```bash
cd client
npm install or yarn
npm start or yarn start
```

- Up server

```bash
cd api
npm install or yarn
npm run json-server or yarn run json-server
```
